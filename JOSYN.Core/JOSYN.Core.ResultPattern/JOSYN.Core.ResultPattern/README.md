# JOSYN.Core.ResultPattern

- Stellt eine Implementierung des Result-Patterns bereit für generische- und void-Results.
- Propagiert den Callstack, wenn Result-Fails in einer durchgängigen Result-Aufruf-Chain die Methdode Propagate() nutzen.
- Ist abgesichert duch Nullable Reference Types.
